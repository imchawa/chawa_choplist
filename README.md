Chopshop with tiers

features
- 3 options for scrapping, Automatic, Manual, Let the player choose
- Auto and manual reward modifiers
- Scrap rewards per part min/max and percentage
- Daily streak rewards aka tiers and multiplier 
- Streak reset after x hours of player not active
- Elite status, side hostle car deliver with pd tracker
- Distance checking for player to npc if you know what i mean
- Discord webhook log
- Npc will yell at you
- Inventory overweight check
- If player crashes during stripping, they can resume after relogging(verify with ped). (same for elite vip delivery, no need to verify with ped)
- Hourly or reboot to list refresh (npc will rotate live if hourly)
- Data is saved in a json file, no sql required
- Probably other things just forgot

This was done with qbx, ox_inventory, qbx_police. Edit it to your liking. This gif just shows the list, theres more to it lul.
